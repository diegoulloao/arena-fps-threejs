# arena-fps-threejs

A first-person arena shooter prototype inspired by classic arena FPS games, built with Astro, React, and Three.js.

## Current Features

- First-person controls with pointer lock
- WASD movement system
- Physics-driven player body with `@react-three/cannon`
- Textured ground and sky rendering

## Scripts

Run commands from the project root:

| Command | Description |
| :-- | :-- |
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview production build locally |

## Stack

- Astro
- React
- Three.js (`@react-three/fiber`, `@react-three/drei`)
- Cannon physics (`@react-three/cannon`)
- Tailwind CSS
